# masscannerpy
A Masscanner script made in python for scan server minecraft

# Requirements
- Masscan
- Python
```
apt install masscan
apt install python3
```

# Installation
```
git clone https://github.com/lucadado/masscannerpy.git
cd masscannerpy
python3 
python3 masscan.py
```
